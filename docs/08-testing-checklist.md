# 테스트 체크리스트

이 문서는 발표 전과 통합 테스트 때  
학생팀이 **반드시 직접 눌러봐야 하는 시나리오**를 정리한 문서입니다.

---

## 1. 테스트 전략

### 우선순위
1. 핵심 시나리오가 끝까지 도는지 확인
2. 권한이 올바르게 막히는지 확인
3. 비즈니스 규칙이 지켜지는지 확인
4. 배포 환경에서 실제로 동작하는지 확인

## 11. 수동 E2E 시나리오

### 시나리오 A. 직원 등록부터 로그인까지
1. 관리자 로그인
2. 새 직원 등록
3. 등록된 직원 계정으로 로그인
4. 내 정보 확인

### 시나리오 B. 출근부터 퇴근까지
1. 직원 로그인
2. 출근 클릭
3. 대시보드 반영 확인
4. 퇴근 클릭
5. 월간 근태 목록 반영 확인

### 시나리오 C. 휴가 신청부터 승인까지
1. 직원 로그인
2. 휴가 신청
3. 내 휴가 목록에서 REQUESTED 확인
4. 매니저 로그인
5. 승인 대기 목록에서 승인
6. 직원 계정에서 APPROVED 확인
